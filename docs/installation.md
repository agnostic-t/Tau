# Как запустить Tau на телефоне

> Спойлер: делается это всё через Termux

## Шаги установки:

1) **Установите Termux**

Делается это через Github или F-Droid ([офиц. репозиторий Termux](https://github.com/termux/termux-app))

На странице [релизов](https://github.com/termux/termux-app/releases) вам надо выбрать последнюю сборку и скачать termux-app_...arm64-v8a.apk (если у вас современный смартфон, если не уверены - скачивайте termux-app_...universal.apk)

2) **Настройка Termux**

После установки Termux (или если вы давно не заходили):

> Если будет спрашивать выбор "Y/I/N", пишите `Y` если у вас нет причин для обратного. 

```sh
pkg update
apt upgrade 
```

Это обновит все пакеты, зеркала и репозитории. Далее вам надо установить `go` и `git`:

```sh
pkg install golang git
```

3) **Установка Tau**

После скачивания go и git вам надо установить сам Tau:

```sh
git clone https://github.com/agnostic-t/tau
cd tau
go build -o ~/taucli ./client/main.go
```

Go сам скачает все нужные зависимости и создаст бинарник.

4) **Конфигурация Tau**

О том, как конфигурировать Tau написано в основном [README](../README_RU.md)

Если вам уже кинул кто-то готовый конфиг, то вам надо дать термукс доступ к файлам и скопировать конфиг (подразумевается, что конфиг находится в загрузках и название `config.json`) в домашнюю директорию:

```sh
termux-setup-storage

mkdir -p ~/.config
cp ~/storage/downloads/config.json ~/.config/tau-conf.json
```

5) **Запуск Tau и использование**

После сборки и получения конфигурации вам надо сделать так, чтобы Termux мог работать в фоне (выдается через настройки батареи для приложений, поставьте "Без ограничений"). Иногда Андройд спрашивает данные настройки если нажать "Acquire Wakelock" (для работы в фоне) во всплывающем уведомлении Termux'а

После выдачи разрешния для работы в фоне:

```sh
~/taucli ~/.config/tau-conf.json
```

Тау сам должен запуститься. Теперь у вас локально на устройстве работает SOCKS5 прокси на адресе из конфига (обычно 127.0.0.1:9005). Данный прокси можно сразу добавить через телеграм.

6) **Использование для всей системы**

После запуска Tau можно через сторонние инструменты (например [dumdum](https://github.com/Project-Mandarin/DumDum)) сделать прокси на всю систему.

Сначала вам надо скачать apk со [страницы релизов](https://github.com/Project-Mandarin/DumDum/releases), после установки:

1) Добавьте SOCKS5 прокси:

![[DumDum screenshot]](../assets/dumdum-0.jpg)

2) Установите правильные настройки адреса (Server: 127.0.0.1, Port: 9005):

![[DumDum screenshot]](../assets/dumdum-1.jpg)

Далее вам **необходимо** сделать режим VPN для приложений (Settings -> Apps VPN mode). Крайне советую включить Proxy режим и **только** для тех приложений, которые надо. Пример:

![[DumDum screenshot]](../assets/dumdum-2.jpg)

Telegram не включен сюда, т.к. он имеет собственные настройки прокси

7) **Если Tau отключился или не подключается**

Есть 3 типа проблем (обычно):
- Tau завис/не грузит: может решится просто заходом в приложение Termux и выходом (чтобы он "проснулся")
- В логах Tau множество ошибок `Failed to connect to VPN`. Тут 2 сценария:
  - Либо у вас отключен интернет
  - Либо сервер упал. В таком случае вам надо поменять выбранный сервер в конфиге.
- Tau почему-то отключился (программа не работает): вам надо просто перезапустить его, еще раз прописав `~/taucli ~/.config/tau-conf.json`

**Как поменять выбранный сервер**:

Сначала вам надо установить какой-либо редактор текста (например `micro`):

```sh
pkg install micro
```

Далее вы пишете (открытие конфига):

```sh
micro ~/.config/tau-conf.json
```

У вас откроется текст конфига. Вам надо поменять значение у ключа `selected`. Пример:

Было:
```json
{
  "lproxy": {
    "socks5": "192.168.1.5:9005"
  },
  "selected": "null-xorek",
  "servers": {
...
```

Стало:
```json
{
  "lproxy": {
    "socks5": "192.168.1.5:9005"
  },
  "selected": "null-timeweb",
  "servers": {
```

После изменения нажмите **Ctrl + S** и затем **Ctrl + Q**, перезапустите Tau (остановить можно нажав **Ctrl + C**)
